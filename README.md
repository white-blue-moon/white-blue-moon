## ■ 주요 프로젝트: 던파 경매 판매 알림 시스템  
> 던전앤파이터 경매 아이템 판매 알림 웹서비스 및 관리자 어드민툴

### ■ 개요
- 사용자가 등록한 아이템을 1분 간격으로 경매장에서 모니터링하며, 해당 아이템이 판매되면 Slack DM으로 알림을 전송하는 웹 서비스입니다.
- 일반 사용자를 위한 **홈페이지**와 운영 관리를 위한 **관리자 어드민툴**로 구성되어 있습니다.

### ■ 주요 기능

#### ▶ 사용자 홈페이지
- 관심 아이템 등록 및 **1분 간격 자동 감시**
- Slack 연동을 통한 **개인 DM 알림 수신**

#### ▶ 관리자 어드민툴
- 사용자 권한 및 설정값 관리
- 서비스 점검 상태 관리

### ■ 기술 스택
- **Frontend**: Svelte, HTML/CSS, JavaScript  
- **Backend**: Spring Boot (REST API), Express, MySQL  
- **Infra**: AWS EC2, Nginx, HTTPS, WireGuard VPN  
- **Integration**: Slack Bot API, Cron 기반 경매 데이터 수집
